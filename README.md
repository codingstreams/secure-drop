# Secure Drop

A secure file-sharing platform where files are encrypted, accessible via 
QR code or access code, and automatically deleted after download or time 
expiration.

**Features:**

* End-to-end encryption
* Time-limited access (download or delete)
* QR code or access code sharing for easy distribution
* Simple, intuitive interface for users

**How it works:**

1. Upload your file to Secure-Drop.
2. Receive a unique QR code or access code.
3. Share the QR code or access code with others.
4. Once downloaded, the file will be automatically deleted after a set 
time period (configurable).
